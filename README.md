# NumberGuessingGame

一個簡易的 Java 終端機互動遊戲
系統會隨機產生 1～100 的整數，玩家需要不斷輸入數字來猜，程式會依據你的輸入給出「太大」或「太小」的提示，並逐漸縮小猜測範圍

- 功能介紹：
隨機產生 1~100 的整數作為答案
玩家輸入數字後會立即顯示：
  太大了→更新上限
  太小了→更新下限
持續提示最新區間（例如：30 ~ 78）
猜中後顯示正確答案並結束遊戲

- 程式流程
1. 程式產生答案
2. 玩家開始猜
3. 若猜錯→顯示提示 + 更新區間
4. 重複直到猜中

如何執行：
1. 確保已安裝 JDK 8+
2. 編譯程式：
```bash
javac Game.java
```
3. 執行：
```bash
java Game
```

- 專案目的：
練習 Java 基礎
熟悉 `Random`、`Scanner` 等常用類別
提升迴圈與條件判斷設計能力
作為 GitHub 學習紀錄
